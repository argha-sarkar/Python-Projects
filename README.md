## Installation

Install Python3 on Ubuntu

```bash
  sudo apt install python3.8
```
Check-out Python3 Version 

```bash
  python3 --version
```
After install Python3 then Install pip

```bash
  pip install pip
```
Install Jupyter Notebook

```bash
  pip install jupyter
```
